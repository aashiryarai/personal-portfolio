EchoChamberAI

Real-Time Debate Engine for Social Media Narratives · Build with Gemini Hackathon

An AI-powered application that turns polarized social media conversations into interactive, balanced debates. Users sign in with their X (Twitter) account, select a tweet on a controversial topic, and the system analyzes it using Gemini to generate opposing viewpoints and retrieve relevant content from both sides. Creates a CNN-style debate video using Veo 3.0 and AI-generated anchors. Users can pause the video and ask questions via voice, receiving real-time, in-character responses powered by Gemini and ElevenLabs with answers grounded in source content and video timestamps.

Engineered a full-stack pipeline (Node.js/Express + frontend dashboard) handling real-time analysis, parallel video generation, and seamless user interaction.
